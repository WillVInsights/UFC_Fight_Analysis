Logic for Data Selection¶

Historical Relevance - The analysis focuses on data from January 1, 2018, to December 31, 2021. This period is recent enough to capture relevant trends in UFC data without delving too far back in time.¶

Avoiding Forecasted Data - Data beyond December 31, 2021, was excluded to ensure only historical, actual data points are used, maintaining the analysis's integrity.¶

Sufficient Data for Analysis - Spanning four years, from 2018 to 2021, the dataset provides ample data for robust statistical analysis and modeling.¶

5. Conducting a Dickey-Fuller test for stationarity¶

In [15]:
# Count the number of fights per date
fight_counts = subset_data.groupby('date').size()


# Dickey-Fuller test on fight_counts
from statsmodels.tsa.stattools import adfuller

dftest = adfuller(fight_counts, autolag='AIC')

# Create a summary of the test results
df_output = pd.Series(dftest[0:4], index=['Test Statistic', 'p-value', '#Lags Used', 'Number of Observations Used'])
for key, value in dftest[4].items():
    df_output[f'Critical Value ({key})'] = value

print('Dickey-Fuller Test Results:')
print(df_output)
